# Project Explanation

## Project Title
Hotel Reservation System (DBMS Project)

## Objective
The objective of this project is to design and implement a database system to manage hotel bookings, customers, rooms, and payments.

## Technology Used
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

## Features Implemented
- Customer registration
- Room category and room management
- Reservation system
- Billing and payment tracking
- Trigger-based automation
- Transaction handling
- Role-based access control

## Implementation Steps

1. Created database and tables with constraints
2. Inserted sample data
3. Created views for easier data access
4. Created indexes for performance improvement
5. Created functions for reusable logic
6. Created triggers for automation
7. Implemented transactions for booking system
8. Applied access control using roles

## Final Working
- Booking system works correctly
- Double booking is prevented
- Room status updates automatically
- Audit log tracks changes

## Conclusion
The system successfully demonstrates DBMS concepts like normalization, constraints, transactions, and triggers in a real-world scenario.
